# Python Learning Journey for Embedded Engineers 🚀

Welcome to my Python learning repository!

I'm an **Embedded Software Tech Lead** with **12 years of experience in Automotive Embedded Systems**, primarily working with C. This repository documents my transition to Python while following modern software engineering practices and building production-quality code.

This is not just a collection of Python exercises—it's a structured learning journey focused on applying Python to engineering automation, embedded systems, and AI.

---

## 🎯 Objectives

This repository is designed to:

* Learn Python the Pythonic way
* Build a strong foundation in core Python concepts
* Apply Python to embedded engineering and automation
* Learn professional Git and GitHub workflows
* Prepare for AI, Machine Learning, and engineering productivity tools
* Build a portfolio of clean, well-structured Python projects

---

## 📚 Learning Approach

Every topic in this repository follows a consistent workflow:

* One topic = One Python file
* One topic = One Git commit
* One feature = One Git branch
* Follow Pythonic coding practices
* Compare Python concepts with C whenever applicable
* Use engineering and embedded systems examples wherever possible

The goal is not only to learn Python syntax, but also to develop professional software engineering habits.
